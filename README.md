# chipseq-pipeline
A standardized and reproducible ChIP-seq analysis pipeline used in Korkmaz Lab

This repository documents the analysis workflow and provides example references to publicly available datasets for demonstration and documentation purposes.

## Scope

This repository is intended to:

* Document the ChIP-seq analysis workflow used in Korkmaz Lab
* Provide references to publicly available datasets for demonstration purposes
* Support reproducible and transparent analysis practices
* Serve as a reference for training and routine analyses

---

## What This Repository Does NOT Contain

This repository does **not** include:

* Raw or processed sequencing data
* Results, figures, or outputs from unpublished studies
* Sample names or metadata from internal experiments
* Any proprietary or sensitive data

All example datasets are referenced only by public accession numbers
(e.g. GEO GSE IDs).

---

## Repository Structure

* `pipeline/` – Documentation of analysis steps
* `config/` – Example configuration and sample templates
* `docs/` – Tutorials, references, and additional documentation
* `data/` – References to example public datasets (no data hosted)
